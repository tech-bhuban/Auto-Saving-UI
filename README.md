# 📝 Instant Notes App

A minimalist, full-stack Note-Taking application built with **Node.js** and **Express**. This project features a self-contained UI and a RESTful API, all running from a single server file.

## ✨ Features
- **CRUD Operations**: Create, Read, and Update notes seamlessly.
- **Auto-Save Logic**: Uses `onchange` events to sync your notes with the server instantly.
- **Zero Frontend Files**: The UI is served directly from the backend for maximum simplicity.
- **In-Memory Storage**: Lightweight and fast for demonstration purposes.

## 🛠️ Tech Stack
- **Backend**: Node.js
- **Framework**: Express.js
- **Frontend**: Vanilla JavaScript & HTML5
